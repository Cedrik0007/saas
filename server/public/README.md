# Logo Directory

Place the Indian Muslim Association logo image here.

**File name:** `logo.png`, `logo.jpg`, or `logo.jpeg`

**Supported formats:** PNG, JPG, JPEG

The logo will be displayed in the top left corner of payment confirmation PDFs.

**Recommended size:** 200x200 pixels or higher (will be resized to 60x60 points in PDF)

**Note:** The system will automatically search for the logo in multiple locations:
- `server/public/logo.*`
- `server/assets/logo.*`
- `public/logo.*`
- `assets/logo.*`
